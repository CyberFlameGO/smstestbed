NIST SMS Test Bed: Technical Data Packages

This section contains Technical Data Packages (TDPs) generated in the SMS Test Bed. Please visit the SMS Test Bed Information Page for more information on the test bed. More information on TDPs coming soon.

TDPs Repository Description

Below is an overview with links to the TDPs available in this repository.

    +---d2mi (Design to Manufacturing and Inspection project)
        +---CAD (Computer-aided design files)
        +---CAM (Computer-aided manufacturing files)
        +---G-code (ISO 6983 G-code related to the CAM and Setup Sheets)
        +---Setup Sheets (Setup Sheets related to the CAM and G-code)
    +---README.md (this documentation as a markdown document)
